# Production App Testing & Support Simulation

This repository demonstrates entry-level QA and Technical Support
skills by simulating testing and troubleshooting of a production-style
web application.

## Skills Demonstrated
- Manual Testing & Test Case Design
- API Testing (Postman)
- Basic Automation Concepts
- Linux Commands & Log Analysis
- Incident & Bug Reporting
- SQL Basics
- Understanding of SDLC / STLC

## Project Overview
A sample web application with REST APIs was tested for functional
correctness and failure scenarios. API endpoints were validated,
issues were logged, and root causes were analyzed using logs and
basic system commands.

## Relevance to Role
This project aligns with responsibilities of:
- QA Engineer
- Technical Support Engineer

## Current Status
Initial documentation and structure added.
Test cases, API collections, and incident reports are being
progressively updated.
